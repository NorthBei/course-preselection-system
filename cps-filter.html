網際服務軟體工程
上課地點 INS105,INS105,INS105
學分 3
課程核心能力
教學目標 本課程針對網際服務應用軟體(Web-based Applications)特性，透過原理與實作的結合，引導學生有效運用軟體工程建構Web應用系統。課程主軸分為兩部分，分別為以J2EE為主軸的Web軟體技術(Web technology)與網際服務軟體工程流程(Web-based software engineering process)，希望透過技術與流程之整合以開發高品質網際服務軟體。
先修科目 網路概論、網路程式設計(建議)、Java程式設計(建議)
教材內容 1.Web App Architecture: high-level overview 2.Mini MVC Tutorial 3.Servlet: request AND response 4.Web App: attributes and listeners 5.Conversational state: session management 6.Using JSP + Scriptless JSP 7.JSTL (JavaServer Pages Standard Tag Library) 8.JavaScript for AJAX 9.XML and JSON 10.建立大規模的Web應用系統 11.軟體需求分析 12.新穎Web 技術報告 ＋ 專案提案 13.Web軟體設計 + StarUML實習 14.需求分析文件報告 15.Web軟體測試 + Mantis實習 (Bug Tracking) 16.Selenium實習 (功能測試) + JMeter (效能測試)
教學方式 1.上課講義講解與實例解釋2.專題製作
參考書目 "Head First Servlets & JSP, 2nd Version", by Bryan Basham, Kathy Sierra and Bert Bates, O"Reilly, 2008 "軟體工程"，李允中著，高立圖書有限公司，2009
教學進度 1.Web App Architecture: high-level overview 2.Mini MVC Tutorial 3.Servlet: request AND response 4.Web App: attributes and listeners 5.Conversational state: session management 6.Using JSP + Scriptless JSP 7.JSTL (JavaServer Pages Standard Tag Library) 8.JavaScript for AJAX 9.XML and JSON 10.建立大規模的Web應用系統 11.軟體需求分析 12.新穎Web 技術報告 ＋ 專案提案 13.Web軟體設計 + StarUML實習 14.需求分析文件報告 15.Web軟體測試 + Mantis實習 (Bug Tracking) 16.Selenium實習 (功能測試) + JMeter (效能測試)
評量方式 期中考 Homework 期末專案(新穎技術報告+網際服務應用規劃)
